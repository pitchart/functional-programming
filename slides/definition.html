Programmation fonctionnelle


un paradigme de programmation de type déclaratif qui considère le calcul en tant qu'évaluation de fonctions mathématiques
Comme le changement d'état et la mutation des données ne peuvent pas être représentés par des évaluations de fonctions, la programmation fonctionnelle ne les admet pas, au contraire elle met en avant l'application des fonctions, contrairement au modèle de programmation impérative qui met en avant les changements d'état

Aborder la résolution de problème de façon différente, en explicitant le quoi plutôt que le comment

Imperative programming

$cities = [];
foreach ($users as $user) {
    if ($user->getAddresses() !== null) {
        foreach ($user->getAddresses() as $address) {
            if ($address->isActive()) {
                $cities[] = new City($address->getZip(), $address->getCity());
            }
        }
    }
} // WARNING: cyclomatic complexity

VS Declarative programming

SELECT zip, city
FROM address
INNER JOIN user
    ON address.user = user.id
WHERE address.is_active = 1
